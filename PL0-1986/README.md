# PL/0 1986
**PL/0** is a programming language intended as an educational language similar to, but simpler than, **Pascal** (a general-purpose programming language). An extended **PL/0** language was introduced, along with a new Compiler-Interpreter system, in the book *Compilerbau* by *Niklaus Wirth* (1986).
